# qt-runway — 现金流推演实验

**回款周期不确定 vs 固定支出** 的实验工具。

## 场景

一家典型中小企业：
- 当前现金 **50万**
- 月固定支出 **30万**（薪资为主）
- 预期回款三笔，共 **130万**

## 推演结果

| 回款延迟 | 可维持月数 | 结论 |
|----------|-----------|------|
| 正常 | 6个月 | 健康 |
| 延迟30天 | 6个月 | 仍可运转 |
| 延迟60天 | **2个月** | 第2个月断裂 |

核心发现：**回款延迟存在一个临界点，越过它之后断裂来得非常快。** 延迟从30天到60天，跑道从6个月骤降到2个月——因为支出是连续的，而回款是离散的。

## 用法

```bash
./runway.py --demo        # 演示场景
./runway.py               # 交互模式，输入自己的数据
```
