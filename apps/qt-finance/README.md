# qt-finance — 财务决策推演 CLI

最小演示：将模糊的财务直觉拆解为结构化的决策推理链路。

## 用法

```bash
./eval.py "买一年还是买三年？三年省200块"
```

输出：

```
场景: 软件订阅期限选择
金额: ~200元（三年优惠）
适用规则: 小金额决策（≤千元，秘书自决范围）
隐含假设:
  - 未来1年内业务变动概率 > 50%
  - 200元不值得牺牲灵活性
推演:
  方案A（一年）: 成本X，灵活性高，可随时调整
  方案B（三年）: 成本X-200，锁定期3年，退出成本未知
结论: 选一年 → 符合规则"小金额决策，灵活性 > 小额优惠"
```

## 设计思路

`eval`（Python）用关键词匹配已知规则（来自 `rules.md`），演示"推演链路"的格式。替换为 AI 后端后，只需将输入和 `rules.md` 一起喂给模型，即可处理任意决策场景。
